Time converter is a project that transforms days to seconds

Original Project Code

import java.util.Scanner;

public class Program {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int days = scanner.nextInt();

int seconds = days*24*60*60;

System.out.println(seconds);


}
}

Loan Calculator

Another of Sololearn java course challenges. Loan Calculator is a project that ask you to calculate the tax that you will need to pay over a loan (Without decimals!).


Output:

Original Project Code

import java.util.Scanner;
public class Program
{
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int amount = scanner.nextInt();

int tax = 10;
int time = 3;
while(time>0){
amount = amount-(amount/tax);
time--;
}
System.out.println(amount);



}
}
